MongoFB

The last NoSQL database you’ll ever need.

MongoFB is an open-source document database, featuring all the SQL commands you know and love. It’s written fully in everyone’s favorite language, JavaScript, and takes advantage of some of the most advanced technology on the face of the planet for data storage. It’s fast, agile, and scalable—all you need is a Facebook account!

Features

Status-Oriented Storage

All your data is stored safely in the most secure, simple, yet powerful schema of all time: Facebook statuses.

Full Index Support

We index the entire table for your convenience. You’re welcome.

Querying

Rich document-based queries using a subset of everyone’s favorite SQL commands.

Map/Reduce

Runs an advanced, parallel, distributed algorithm on a single node to process Big Data.

Fast Nonatomic Updates

Atoms are scary and can be used to build bombs. We got rid of them for you; enjoy the contention!
